🔑 Token de GitHub
¿Cómo crear el token?

1. Ir a github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens
2. Click en Generate new token
3. En Repository access, elegir solo el repo ivoTech
4. En Permissions → Contents, elegir Read and Write
5. Generar y copiar el token aquí.

El token se guarda solo mientras tengas el navegador abierto.
